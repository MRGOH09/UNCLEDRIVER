# 🚐 Uncle 载送点名系统

一个用于管理学生接送服务的移动端点名系统，集成飞书多维表格API。

## ✨ 功能特色

- **双向接送管理** - 支持来接(C)和送回(B)两个班次
- **实时状态更新** - 点击切换学生状态（待确认/已上车/缺席）
- **移动端优化** - 响应式设计，触摸友好
- **自动数据同步** - 与飞书多维表格实时同步
- **系统验证** - 内置测试面板，确保功能正常

## 🚀 快速开始

### 部署到Vercel
1. Fork此仓库
2. 在 [Vercel](https://vercel.com) 连接GitHub
3. 选择此仓库进行部署

### 配置
更新 `uncle-rollcall.html` 中的配置信息：
```javascript
const CFG = {
  APP_ID: '你的飞书APP_ID',
  APP_SECRET: '你的飞书APP_SECRET', 
  APP_TOKEN: '你的多维表格TOKEN',
  MASTER_TABLE: '主表ID',
  ROLLCALL_TABLE: '点名表ID'
};
```

## 📱 使用方法

### 基本操作
- 点击学生状态标签切换状态
- 点击电话图标拨打电话
- 点击WhatsApp图标发送消息
- 右下角刷新按钮同步最新数据

### 测试模式
- 双击页面顶部日期区域开启测试面板
- 运行系统验证确保功能正常

## 🔧 系统要求

- 现代移动浏览器
- 网络连接
- 飞书多维表格访问权限

## 📊 数据结构

### 主表字段
- 名字、driver、ADDRESS、电话号码
- MON(C)、TUE(C)...等时间字段

### 点名表字段
- 名字、日期、班次、司机、时间
- 地址、电话、点名时间、状态

## 🛡️ 安全注意

⚠️ **重要**: 当前API密钥直接写在代码中仅用于测试。生产环境请使用环境变量或服务端代理。
